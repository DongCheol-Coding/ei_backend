# 📚 EI Backend

> 이 프로젝트는 회원가입, 소셜 로그인, 강의 수강, 실습 체크, 실시간 채팅 기능 등을 포함한 **교육 플랫폼 백엔드 API**입니다.

---

## 🛠️ 기술 스택

- **Java 21**
- **Spring Boot 3.x**
- **Spring Security + JWT + OAuth2.0**
- **JPA (Hibernate)**
- **Gradle**
- **MapStruct**
- **Lombok**
- **Amazon S3 (이미지 업로드)**
- **Amazon RDS (MariaDB)**
- **GitHub Actions + EC2 (CI/CD)**
